## 📌 Overview
News Flash is a fast, responsive news application that fetches the latest headlines from India and around the world using the **GNews API**.  
It includes search, category filters, pagination, and a clean UI.

---

## 🚀 Features
- 🔎 Search any topic  
- 🇮🇳 Indian + 🌍 International news  
- 🗂 Category filters (Finance, Politics, Tech, Entertainment)  
- 🔄 "Load More" pagination  
- ⚡ Fast API performance  
- 📱 Fully responsive UI  

---

## 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **GNews API**
